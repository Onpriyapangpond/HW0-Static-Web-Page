รู้จัก 22 สมุนไพรในประเทศไทย ใกล้สูญพันธุ์!!

กระทรวงสาธารณสุขได้ประกาศรายชื่อ สมุนไพรไทย จำนวน 22 ชนิด อยู่ใน “ภาวะเสี่ยงต่อการสูญพันธุ์”

เนื่องจากเป็นพนธุ์ไม้ที่เติบโตช้า บางชนิดจำเป็นต้องใช้ ทั้งต้นและราก บางชนิดนำมาใช้บ่อยจนโตไม่ทันใช้งาน ซึ่งมีสมุนไพร 7 ชนิด
ที่จะประกาศคุ้มครอง และเป็นสมุนไพรที่ต้องควบคุม ได้แก่ ถั่วดินโคก เทพทาโร มะตูมนิ่ม มะหาด เร่ว หัวร้อยรู และกระวาน

"โดยหากใครมี สมุนไพรไทย 7 ตัวนี้ อยู่ในครอบครอง ไม่ได้ถือว่ามีความผิด"

เพียงแต่ต้องแจ้งมาทางกระทรวงสาธารณสุข ให้รับทราบว่ามีอยู่เท่าใด เพื่อที่จะจัดทำแผนส่งเสริมการปลูกต่อไป
เว็บไซต์บ้านและสวนจึงอยากนำมาแนะนำให้รู้จักถึง สรรพคุณทางยาของสมุนไพร ทั้ง 22 ชนิด ดังนี้

1.ถั่วดินโคก

2.เทพทาโร

สรรพคุณ : ช่วยลดไข้ในเด็ก ใช้ต้มกับน้ำดื่มเป็นยาบำรุงน้ำนมของสตรี
และใบนำมาฝนทาเป็นยาแก้ฝี

สรรพคุณ : แก้ไข้หวัด แก้ไอ อาการไอเรื้อรัง ตัวร้อน ออกหัด เนื้อไม้ใช้ปรุง
เป็นยาหอมลม รักษาท้องขึ้น อืดเฟ้อ

3.มะตูมนิ่ม

4.มะหาด

สรรพคุณ : คนโบราณจะนำมาทำเป็นยาอายุวัฒนะ เพื่อบำรุงกำลัง บำรุงเพศ
ชลอความแก่ ทำให้ผมหงอกช้า

สรรพคุณ : แก่นมีสรรพคุณเป็นยาแก้ลม เปลือกต้นสดนำมาต้มกับน้ำ
กินเป็นยาแก้ไข้

5.เร่ว

6.หัวร้อยรู

สรรพคุณ : ช่วยแก้อาการหืด,ไอ,คลื่นไส้อาเจียน รับประทานอาหารไม่ได้
ผลช่วยแก้เสมหะในลำคอ เมล็ดช่วยกัดเสมหะ

สรรพคุณ : ช่วยบำรุงหัวใจ หัว ใช้ตำกินเป็นยาขับพยาธิ

7.กระทุ่มนา

8.ขันทองพยาบาท

สรรพคุณ : ช่วยลดความดันโลหิต ใบและเปลือกต้นต้มกินเป็นยาแก้ไข้
ใช้อมกลั้วคอแก้อาการอักเสบของเยื่อเมือกในปาก

สรรพคุณ : ใช้เป็นยาบำรุงเหงือก รักษาเหงือกอักเสบ ทำให้ฟันเหงือกแข็งแรง
ขับระดูร้าย แก้โรคตับ และถอนพิษ

9.จุกโรหินี

10.ชะเอมไทย

สรรพคุณ : ผลนำมาต้มกับน้ำดื่ม จะช่วยลดระดับน้ำตาลในเลือดได้
ช่วยแก้อาการอ่อนเพลีย ช่วยบำรุงกำลัง รากนำมาเคี้ยวกับพลูช่วยแก้อาการไอ

สรรพคุณ : เปลือกต้นใช้ต้มกับน้ำดื่มช่วยแก้อาการไอ หากอาการไม่ดีขึ้น
ให้รับประทานติดต่อกัน 2-4 วัน ช่วยขับเสมหะ แก้น้ำลายเหนียว

11.ซิงซี่

12.ตับเต่า

สรรพคุณ : ใบนำมาต้มดื่มเป็นยาแก้ไข้ รากใช้เป็นยาขับปัสสาวะ
ช่วยทำให้มดลูกเข้าอู่

สรรพคุณ : เปลือกใช้เป็นยารักษาโรครำมะนาด น้ำต้มจากแก่น
และรากมีสรรพคุณเป็นยาบำรุงปอด

13.นางแย้มป่า

14.ปลาไหลเผือก

สรรพคุณ : รากนางแย้มป่าใช้ต้มเป็นยาแก้ไข้ นอกจากนี้ยังช่วยรักษา
ลำไส้อักเสบ ในตำรายาไทย ใช้รากเป็นยาช่วยขับปัสสาวะ

สรรพคุณ : รากใช้เป็นยาบำรุงกำลัง บำรุงร่างกาย คนเดินป่านิยมกัน
เพราะจะช่วยทำให้ร่างกายแข็งแรง

15.พังคี

16.มะคังแดง

สรรพคุณ : รากพังคีช่วยแก้อาการจุกเสียด แก้ท้องอืด ท้องเฟ้อ ปวดท้อง
รากใช้ตำประคบแก้อาการปวด

สรรพคุณ : เนื้อไม้มีรสเย็นเฝื่อน ใช้ต้มกับน้ำดื่มเป็นยาแก้เลือดลมเดินไม่สะดวก
แก้พิษโลหิตและน้ำเหลือง เปลือกต้นใช้ตำพอกแผลสดเพื่อห้ามเลือด

17.สะค้าน

18.สารภีป่า

สรรพคุณ : เครือ ใช้ประกอบอาหารช่วยเพิ่มรสเผ็ด ลำต้นใช้ใส่แกง
ช่วยให้มีกลิ่นหอม

สรรพคุณ : ดอกมีสรรพคุณเป็นยาบำรุงหัวใจ ดอกมีสารช่วยขยายหลอดลม
และช่วยขับลม เปลือกและดอกมีสรรพคุณเป็นยาแก้บิด

19.อบเชยไทย

20.เฉียงพร้านางแอ

สรรพคุณ : เป็นยาร้อนออกฤทธิ์ต่อไต ม้าม และกระเพาะปัสสาวะ ใช้เป็นยาบำรุง
ทำให้ร่างกายอบอุ่น ช่วยกระจายความเย็นในร่างกาย ทำให้เลือดหมุนเวียนดี

สรรพคุณ : ลำต้นใช้ต้มน้ำดื่มช่วยบำรุงร่างกาย ทำให้เจริญอาหารหรือใช้เป็น
ยาเจริญอาหารสำหรับสตรีหลังคลอดบุตร

21.เถาเอ็นอ่อน

22.เปราะหอม

สรรพคุณ : เป็นยาเย็น มีพิษ ออกฤทธิ์ต่อหัวใจและตับ ใช้เป็นยาฟอกเลือด
ขับลมในลำไส้และในกระเพาะอาหาร ทำให้ผายลมและเรอ ช่วยแก้
อาการจุกเสียดแน่นท้อง

สรรพคุณ : ใช้เป็นยาแก้ปวดศีรษะ คลายเครียด ใช้เป็นยาบำรุงหัวใจ บำรุงประสาท
ด้วยการใช้หัวผสมลงในยาหอม หัวเปราะหอมนำมาต้มหรือชงกิน จะช่วย
ในการนอนหลับได้เป็นอย่างดี และยังช่วยลดความเครียดได้อีกด้วย



“ตามปกติแต่ละจังหวัดจะมีการจดทะเบียนรายชื่อสมุนไพรที่มีอยู่ในจังหวัดนั้นๆ แบ่งออกเป็น 3 ประเภท
คือ 1. สมุนไพรเพื่อการศึกษา 2. สมุนไพรพืชเศรษฐกิจ และ 3. สมุนไพรเสี่ยงอาจสูญพันธุ์
โดยจะมีคณะผู้เชี่ยวชาญเข้ามาจัดทำข้อมูลของสมุนไพรทั้ง 3 ประเภท จากนั้นได้เลือกออกมาเป็น 22 ชนิด
ที่เสี่ยงอาจจะสูญพันธุ์ เนื่องจากเป็นสมุนไพรที่เติบโตช้า เมื่อจะใช้ต้องใช้ทั้งต้น บางชนิดใช้ทั้งราก และต้องใช้บ่อยจึงจะสามารถรักษาอาการได้”
